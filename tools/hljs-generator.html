Highlight JS HTML Generator

This uses HighlightJS to produce styled HTML code. It doesn't work incredibly, but it is much better than doing it manually. A much more reliable way to do this is to copy code from VSCode, which will be formatted and coloured appropriately. This is more intended for when you want your code output to use CSS colouring, instead of the coloured spans that come out of VSCode.
